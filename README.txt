1. Install spacy on windows
The pre-processing involves lemmatization using spaCy. Please install spaCy and download the English model.

1.1 Operating on Unix/Linux, macOS/OS X and Windows over conda
$ conda install -c conda-forge spacy
$ conda install -c conda-forge spacy-lookups-data
$ python -m spacy download en_core_web_sm

1.2 Operating on Unix/Linux, macOS/OS X and Windows over pip
$ pip install -U spacy
$ pip install -U spacy-lookups-data
$ python -m spacy download en_core_web_sm

1.3 For more about how to use the spaCy please visit:
https://spacy.io/usage

2. The program is designed to run on Jupyter Notebook. Results for models can be found by running 'accuracies'.

3. Deprecation Warning may occur due to the implementation of .ix, which still works fine.

4. Explanations of some abbreviations:
mnb: multinomial naive bayes
lgr: logistic regression
mnb_(1,1): mnb on unigram representation
mnb_(1, 2): mnb on unigram and bigram representation
lgr_(1, 1): lgr on unigram representation
lgr_(1, 2): lgr on unigram and bigram representation

5. Four csv files containing the test predictions in Kaggle format can be generated by the program Test predictions, which is turned off
by assigning False to 'get_csv' in the first cell of '2. The test phase'. 
If you would like to generate the csv files, please change the 'get_csv' to True.
